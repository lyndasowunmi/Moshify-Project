# MOSHIFIED WEBSITE MOCK UP

A website that mocks all of the features, animations, and components of moshified.com

Features:
- Built using HTML, CSS, and JavaScript
- Mobile-responsive
- The website includes;

  * Navigation bar
  * Hero block
  * Domain block
  * Cards & Plans
  * Features block
  * Grids
  * Media Objects block
  * Testimonials block
  * Showcase block
  * Footer
